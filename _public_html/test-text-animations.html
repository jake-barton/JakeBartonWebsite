TEXT ANIMATIONS

1. ROTATING TEXT

Perfect for hero sections and highlighting multiple skills or roles. Words smoothly transition in and out.

I AM A
<span data-rotating-text="WORD1, WORD2, WORD3" data-duration="2000"></span>

2. TYPING TEXT

Simulates a typewriter effect with a blinking cursor. Great for attention-grabbing headers.

<span data-typing-text="TEXT1, TEXT2, TEXT3"></span>

3. GLITCH TEXT

Cyberpunk-style glitch effect on hover. Perfect for interactive elements and modern aesthetics.

HOVER OVER ME
<span data-glitch-text>YOUR TEXT</span>

4. REVEAL TEXT (Scroll Down)

Text animates into view when scrolled into viewport. Supports multiple directions.

REVEAL FROM BOTTOM
REVEAL FROM RIGHT
REVEAL FROM LEFT
<div data-reveal-text data-reveal-direction="up" data-reveal-delay="0">TEXT</div>

5. SPLIT TEXT ANIMATION

Each letter animates individually with a stagger effect. Very eye-catching for headings.

CREATIVE PORTFOLIO
<div data-split-text data-split-animation="slide">YOUR TEXT</div>

6. GRADIENT TEXT

Animated gradient that flows through the text. Perfect for making text stand out.

GRADIENT ANIMATION
<div data-gradient-text="#color1, #color2, #color3">YOUR TEXT</div>

HOW TO USE

STEP 1: Include the script in your HTML

<script src="assets/js/text-animations.js" defer></script>

STEP 2: Add data attributes to your HTML elements

<span data-rotating-text="WORD1, WORD2, WORD3"></span>

STEP 3: That's it! The animations will initialize automatically.

Available Attributes:

  • data-rotating-text="word1, word2, word3" - Rotating words effect
  • data-typing-text="text1, text2, text3" - Typewriter effect
  • data-glitch-text - Glitch effect on hover
  • data-reveal-text - Reveal on scroll
  • data-split-text - Letter-by-letter animation
  • data-gradient-text="color1, color2, color3" - Animated gradient
